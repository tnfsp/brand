# 靈感挖掘系統 - 快速開始

> 如何從你的日常筆記中挖掘內容靈感

---

## 🚀 使用流程

### 一條指令，完成所有步驟：

```bash
/inspire
```

**這會自動執行**：
1. ✅ 從 murmur.wilsonchao.com 抓取最新筆記
2. ✅ 分析筆記中的情感信號、故事、觀察
3. ✅ 產出 3-5 個結構化內容點子
4. ✅ 評分並排序（高/中/長期優先級）

---

## 📋 完整步驟說明

### Step 1: 執行靈感挖掘
```bash
/inspire
```

### Step 2: 等待自動抓取
你會看到：
```
🚀 Murmur 筆記抓取器
📡 正在從 https://murmur.wilsonchao.com 抓取內容...
✅ 筆記已儲存到: [檔案路徑]
📊 抓取字數: XXX 字
```

### Step 3: Agent 自動分析
Inspiration Agent 會：
- 讀取抓取的筆記
- 識別情感信號（困惑、感動、反思、觀察）
- 識別故事性片段（場景、對話、轉折）
- 提煉 3-5 個內容點子

### Step 4: 查看靈感報告
你會收到類似這樣的報告：

```markdown
# 靈感挖掘報告 - 2025-12-12

## 🔥 高優先級點子（立即可用）

### 點子 #1：為什麼心臟外科醫師脾氣特別「暴躁」？
來源筆記：2025-12-09 凌晨五點日出
核心概念：從「凌晨五點值班、八點開刀」理解「暴躁」背後的真相
評分：⭐⭐⭐⭐⭐ 共鳴度 | ⭐⭐⭐⭐ 獨特性 | ⭐⭐⭐⭐ 可執行性

### 點子 #2：...
[其他點子]

## ⭐ 中優先級點子（需補充素材）
[...]

## 💡 長期追蹤主題
[...]
```

### Step 5: 選擇並執行
從報告中選擇你感興趣的點子：
```bash
# 如果要做 IG 貼文
/ig

# 如果要寫 Blog 文章
/blog
```

## 💡 建議的五個內容點子

> 基於《沒有姓名的值班機器》的風格，這些點子已經準備好，你可以直接使用

### 點子 #1：當「抱歉」變成口頭禪 ⭐⭐⭐⭐⭐
**核心概念**：學弟妹打電話時每句話都說「抱歉」，即使沒做錯事。探討為什麼我們習慣性道歉？這是禮貌還是不覺得自己有資格佔用別人的時間？

**可能開場**：
> 最近在值班時注意到一件事：學弟妹打電話來會診時，幾乎每一句話都會說「抱歉」。
>
> "抱歉學長，可以請問一下..."
> "抱歉打擾您..."
>
> 什麼時候開始，我們連正常的工作溝通，都需要先道歉？

**適合形式**：IG 貼文 + Blog 文章
**主題類型**：個人成長（橘色系）

---

### 點子 #2：下班後，我們還是醫師嗎？ ⭐⭐⭐⭐
**核心概念**：聚會時被問「醫師，這個症狀是不是...」探討醫師的專業身份與個人身份的界線。下班後脫下白袍，我們還是"醫師"嗎？

**可能開場**：
> 上週末和朋友聚餐，聊到一半，朋友突然說：「對了，我最近有個症狀，你幫我看看...」
>
> 我笑著回答，但心裡想的是：我現在不是醫師，我只是你的朋友。

**適合形式**：Blog 文章為主
**主題類型**：個人成長（橘色系）

---

### 點子 #3：第一次被病人叫「醫師」的那一刻 ⭐⭐⭐⭐⭐
**核心概念**：回憶第一次被病人叫「醫師」時的心情：興奮？惶恐？不真實？從「學生」到「醫師」的身份轉換。

**可能開場**：
> 還記得第一次被病人叫「醫師」的那一刻嗎？
>
> 我記得。那是我當實習醫師的第一週，一位阿伯看著我說：「醫師，謝謝你。」
>
> 那一刻，我楞了一下。我嗎？我是醫師了嗎？

**適合形式**：IG 貼文（故事型）+ Blog 擴寫
**主題類型**：故事 + 個人成長（橘色系 A）

---

### 點子 #4：我們為什麼不敢說「我不知道」？ ⭐⭐⭐⭐
**核心概念**：醫療現場中，「不知道」是禁忌。會診時被問問題，即使不確定也要給個答案。但假裝知道，風險更大。承認無知，才是專業的開始。

**可能開場**：
> 「這個病人的診斷是什麼？」
>
> 學弟被問到時，我看見他楞了一下，然後開始猜測：「可能是...或許是...」
>
> 但什麼時候，承認無知變成了一種恥辱？

**適合形式**：IG 貼文 + Blog 文章
**主題類型**：個人成長（橘色系）

---

### 點子 #5：當學長姐不再記得你的名字 ⭐⭐⭐⭐
**核心概念**：與《沒有姓名的值班機器》相反的角度——被遺忘的感受。曾經一起值班的學長姐，見面時已經不記得你。在醫院裡，我們是「可替換的」嗎？

**可能開場**：
> 上週在醫院走廊上遇到當年的學長，我笑著打招呼：「學長好！」
>
> 他愣了一下，禮貌地笑笑，但我看得出來——他不記得我了。

**適合形式**：IG 貼文 / Blog 文章
**主題類型**：個人成長（橘色系）
