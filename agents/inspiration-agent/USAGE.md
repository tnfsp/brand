# Inspiration Agent 使用指南

完整的靈感挖掘工作流程

---

## 🚀 快速開始

### 方法 1：從 Murmur 自動抓取（推薦）

```bash
# 1. 抓取最近的 Murmur 筆記
npm run fetch:murmur

# 2. 啟動 Inspiration Agent 分析
/inspire
```

Agent 會自動讀取剛抓取的筆記，並提供結構化的靈感報告。

---

### 方法 2：手動提供筆記片段

```bash
# 1. 啟動 Inspiration Agent
/inspire

# 2. 貼上你想分析的筆記
[貼上 1-3 則最近的筆記或想法]

# 3. Agent 會分析並提供內容點子
```

---

### 方法 3：從 Telegram（需要 Bot 權限）

```bash
# 如果 Bot 是頻道管理員
npm run fetch:telegram

# 然後使用 /inspire 分析
/inspire
```

---

## 📊 完整工作流程

```
1. 定期抓取筆記
   ↓
2. Inspiration Agent 分析
   ↓
3. 產出靈感報告（高/中/長期優先級）
   ↓
4. Wilson 篩選感興趣的點子
   ↓
5. 使用 /ig 或 /blog 指令執行創作
   ↓
6. 發布內容
```

## 💡 實際範例

### 範例 1：從 Murmur 筆記挖掘

**原始筆記**（來自 2025-12-09）：
> 又是凌晨五點的日出，明天八點還有Open heart，難怪心臟外科的脾氣都不好
>
> 每天睡眠不足，脾氣是要怎麼好！

**Inspiration Agent 分析**：

```markdown
## 點子 #1：為什麼某些科別的醫師脾氣特別「暴躁」？

### 📋 基本資訊
- 來源筆記：2025-12-09 凌晨五點日出
- 原始情緒：無奈、疲憊、理解
- 內容支柱：個人成長 / 醫療現場
- 建議形式：IG + Blog

### 💡 核心概念
觀察到心臟外科醫師常被說「脾氣不好」
但當你理解他們的工作節奏：凌晨五點值班、八點開刀、長期睡眠不足
就會明白這不是「脾氣不好」，而是「人性的極限」

延伸：
- 為什麼我們容易標籤某些科別的醫師？
- 醫療現場的「暴躁」背後的真相
- 如何在極限中保持人性？

### 🎯 可能的開場
> 「心臟外科醫師脾氣都很差。」
>
> 這是我實習時最常聽到的話。但有一天，我凌晨五點在醫院看到日出時，突然理解了。
>
> 他們剛值完班，八點還要開 Open heart。每天睡不到三小時，脾氣是要怎麼好？

### ⭐ 評分
- 共鳴度：⭐⭐⭐⭐⭐ (5/5) - 很多人有刻板印象
- 獨特性：⭐⭐⭐⭐ (4/5) - 從同理角度切入
- 可執行性：⭐⭐⭐⭐ (4/5) - Wilson 有親身經驗
```

**後續行動**：
- Wilson 決定採用這個點子
- 使用 `/ig` 或 `/blog` 指令開始創作
- Content-Framing Agent 進一步發展架構

### 範例 2：從 Readwise 摘要挖掘

**原始筆記**：
> 學習，是為了替那些『你尚未被想過的東西』預留空間。
>
> Note: 讀書有趣的地方是，看到那些你未曾想過的事物！

**Inspiration Agent 分析**：

```markdown
## 點子 #2：醫學教育中，我們學的是「知識」還是「思考」？

### 💡 核心概念
醫學教育常強調「記住」知識
但真正重要的是「預留空間給未知」
當我們太依賴既有知識，反而無法看見新的可能性

### 🎯 可能的開場
> 實習時，老師問我：「這個病人可能是什麼診斷？」
>
> 我背了一堆教科書，但老師說的診斷，我完全沒想過。
>
> 那一刻我才明白：學習不是為了「知道」，而是為了「看見更多可能性」。

### ⭐ 評分
- 共鳴度：⭐⭐⭐⭐ (4/5)
- 獨特性：⭐⭐⭐⭐⭐ (5/5) - 結合閱讀與醫學教育
- 可執行性：⭐⭐⭐ (3/5) - 需要更多具體故事
```

**後續行動**：
- 標記為「中優先級」
- 持續觀察相關經驗
- 累積足夠素材後再創作

## 🔄 建議頻率

### 每週一次（推薦）
```bash
# 每週日晚上或週一早上
npm run fetch:murmur
/inspire
```

**好處**：
- 及時捕捉新鮮的想法
- 不會累積太多筆記（容易分析）
- 保持內容產出節奏

### 隨時使用
當你：
- 感到靈感枯竭
- 想不到寫什麼
- 看到特定筆記想深入探討

可以隨時：
```bash
/inspire
[貼上特定筆記]
```

## 💡 使用技巧

### 技巧 1：關注情緒信號
當筆記中出現：
- **困惑**："為什麼..."、"不懂..."
- **感動**："突然覺得..."、"那一刻..."
- **無奈**："又是..."、"難怪..."
- **反思**:"也許..."、"我在想..."

→ 這些通常是好的內容起點

### 技巧 2：識別重複主題
如果某個主題在不同時間點重複出現：
- 睡眠不足
- 醫病溝通
- 學習方法
- 工作壓力

→ 代表這是你真正關心的議題，值得深入寫

### 技巧 3：結合時事或經驗
將筆記中的想法與：
- 最近的臨床經驗
- 正在輪訓的科別
- 社會熱門議題

結合起來，會更有時效性和共鳴

### 技巧 4：不急著立刻執行
- 高優先級點子：立即執行（素材充足）
- 中優先級點子：觀察 2-4 週（累積素材）
- 長期追蹤主題：持續記錄（可能成為系列）

## 📝 注意事項

### ✅ 適合分析的筆記
- 有情感張力的觀察
- 讓你困惑或感動的事件
- 臨床現場的故事或對話
- 個人成長的領悟
- 對體制或文化的反思

### ❌ 不適合分析的筆記
- 純粹的待辦事項
- 太私人的情緒宣洩
- 沒有共鳴潛力的內容
- 過於瑣碎的日常記錄

### 🔒 隱私保護
- `murmur-notes/` 和 `telegram-notes/` 已加入 `.gitignore`
- 這些筆記不會被提交到 Git
- Inspiration Agent 分析後的報告可以選擇是否保存
- 如果筆記涉及病人隱私，記得匿名化或略過

## 🔧 故障排除

### 問題 1：無法抓取 Murmur 筆記
```bash
# 檢查網址是否正確
curl https://murmur.wilsonchao.com

# 或手動從網頁複製筆記
# 然後使用 /inspire 並貼上內容
```

### 問題 2：Telegram 抓取失敗
原因：Telegram Bot API 需要 Bot 是頻道管理員

解決方案：
1. 在 Telegram 頻道設定中，將 Bot 設為管理員
2. 或直接使用 `npm run fetch:murmur`（更簡單）
3. 或手動複製筆記內容

### 問題 3：分析結果不理想
- 確認筆記內容是否足夠豐富
- 嘗試提供更多上下文
- 或手動補充你的想法給 Agent
