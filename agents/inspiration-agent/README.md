# Inspiration Agent - 靈感挖掘助手

從 Wilson 的日常筆記中挖掘內容靈感的 AI Agent。

---

## 🎯 功能

### 自動靈感挖掘
- 從 murmur.wilsonchao.com（Telegram 同步網頁）抓取最近筆記
- 分析筆記中的情感、觀察、故事、反思
- 提煉成結構化的內容點子
- 評分並排序（共鳴度 / 獨特性 / 可執行性）

### 智能分析
識別以下類型的筆記：
- ✅ 有情感張力（喜悅、困惑、感動、無奈）
- ✅ 觀察到的現象（醫療現場、社會、人性）
- ✅ 個人成長相關（學習、改變、領悟）
- ✅ 引發思考的問題
- ✅ 有故事性的片段（場景、對話、轉折）

### 產出格式
- **高優先級點子**：立即可用，素材充足
- **中優先級點子**：有潛力，需補充素材
- **長期追蹤主題**：持續觀察，可能未來發展

---

## 🚀 使用方式

### 方法 1：使用 CLI Slash Command（推薦）

```bash
/inspire
```

Inspiration Agent 會自動：
1. 抓取 murmur.wilsonchao.com 最近 7-14 天的筆記
2. 分析並提煉靈感
3. 產出結構化報告

### 方法 2：手動啟動

在 Claude Code 中：
1. 讀取 `agents/inspiration-agent/agent-prompt.md`
2. 使用 WebFetch 抓取 murmur.wilsonchao.com
3. 依照 agent-prompt.md 的流程分析

### 方法 3：提供筆記片段

如果有特定筆記想分析：
```
/inspire

[然後貼上筆記內容]
```

## 🔄 工作流程

```
1. 抓取筆記
   ↓
2. 初步篩選（標註 🔥⭐💡❌）
   ↓
3. 主題提煉（結構化點子）
   ↓
4. 評分排序
   ↓
5. 產出報告
   ↓
6. Wilson 篩選 → 轉交對應 Agent 執行
```

## 💡 使用技巧

### 定期執行（建議）
- **每週一次**：回顧過去一週的筆記
- **靈感枯竭時**：主動挖掘過去筆記
- **特定主題**：搜尋相關關鍵字的筆記

### 最佳實踐
- ✅ 定期執行，累積靈感庫
- ✅ 不急著立刻執行所有點子（選擇最有感的）
- ✅ 持續追蹤長期主題（可能未來會有更多素材）
- ✅ 保存原始筆記（未來可以重新分析）

### 進階用法
- 結合不同時期的筆記（找出主題演變）
- 分析情緒波動（什麼時候最有創作靈感？）
- 追蹤重複出現的主題（這些可能是核心關注點）

## 📝 注意事項

### ✅ Agent 會做的
- 尊重原始筆記的情感與語氣
- 保持 Wilson 的真實性
- 提供多個方向選擇
- 標註素材充足度

### ❌ Agent 不會做的
- 曲解筆記原意
- 過度推銷某個點子
- 忽略負面情緒（這些也是好素材）
- 建議過於私密的內容

## 🔧 自訂設定

### 調整分析範圍
預設：最近 7-14 天
可調整為：
- 最近一週（快速靈感）
- 最近一個月（深度挖掘）
- 特定時期（如：實習期間、特定科別輪訓）

### 調整評分權重
預設：共鳴度 = 獨特性 = 可執行性
可調整為：
- 優先共鳴度（追求受眾反應）
- 優先獨特性（追求獨特觀點）
- 優先可執行性（快速產出）
