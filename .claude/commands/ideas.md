# 內容點子庫管理

協助管理 `content-hub/ideas-pool.md` 的內容點子。

## 功能

### 1. 檢視點子庫
- 列出目前的內容點子
- 依支柱分類（醫學/故事/個人成長）
- 顯示狀態（待使用/進行中/已發布）

### 2. 新增點子
- 加入新的內容點子到點子庫
- 標記支柱、適合渠道（IG/Blog）、優先順序

### 3. 更新狀態
- 標記點子為「進行中」或「已發布」
- 記錄發布日期與連結

### 4. 補充細節
- 為現有點子補充更多資訊
- 加入參考資料或靈感來源

## 內容三大支柱

### 醫學（藍色系）
- 常見疾病科普
- 醫學迷思破解
- 就醫指南、預防保健
- 用藥安全

### 故事（橘色系 A）
- 臨床故事（匿名化）
- 行醫心得
- 醫病關係
- 個人經歷

### 個人成長（橘色系 B）
- 學習心得
- 個人反思
- 習慣養成
- 跨領域學習
