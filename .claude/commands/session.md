# Session Log 更新

協助更新 `SESSION-LOG.md`，記錄本次 CLI 對話的工作內容。

## 根據 CLAUDE.md 的規範

每次使用 Claude Code CLI 時，都必須記錄：

### 1. 本次目標
這次工作的主要目標是什麼？

### 2. 完成事項
- 列出完成的任務
- 使用 checkbox 格式

### 3. 變更記錄
- 新增的檔案
- 修改的檔案
- 說明變更目的

### 4. 遇到的問題與解決方式
如有問題，記錄如何解決

### 5. 需要 Leader 決策的事項
標註待決策的問題

### 6. 下次待辦事項（Next Steps）
列出下次要做的事

### 7. Git Commit Message
記錄本次的 commit 訊息

## Session Log 範本

```markdown
## Session #XXX - YYYY-MM-DD

### 📌 本次目標
[ 目標描述 ]

### 💬 討論內容
[ 討論摘要 ]

### ✅ 完成事項
- [x] 任務 1
- [x] 任務 2

### 📝 檔案變更記錄
#### 新增檔案
- `path/to/file.md` - 說明

#### 修改檔案
- `path/to/file.md` - 修改內容

### ⏭️ 待辦事項
- [ ] 待辦 1
- [ ] 待辦 2

### 🎯 下次重點
[ 下次要處理的重點 ]

### Git Commit
git commit -m "..."
```
