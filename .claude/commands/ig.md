# IG 內容生產流程

你現在負責 **Wilson Chao IG 貼文的完整生產流程**。

這個 command 會整合所有 IG 相關的 Agents，從內容框架到最終交付。

## 工作流程

### Phase 1: 內容框架（Content-Framing）
參考 `agents/content-framing/agent-prompt.md`
- 從點子庫選擇主題
- 產出內容框架
- 確認適合 IG（三張圖輪播）

### Phase 2: 三圖架構設計（IG-Ideation）
參考 `agents/ig-agents/ig-ideation-agent.md`
- 將框架細化成三張圖的具體架構
- 每張圖的標題、重點、視覺元素
- 確保敘事邏輯流暢

### Phase 3: 文案撰寫（IG-Copywriting）
參考 `agents/ig-agents/ig-copywriting-agent.md`
- 撰寫每張圖的文案
- 撰寫 Caption（貼文說明文字）
- 加入標籤（8-12 個）

### Phase 4: 視覺設計指示（IG-Visual）
參考 `agents/ig-agents/ig-visual-agent.md`
- 選擇色系（藍色醫學/橘色故事/橘色成長）
- 選擇 Canva 模板
- 產出詳細設計指示

### Phase 5: 最終整合（Content-Creation）
參考 `agents/ig-agents/content-creation-agent.md`
- 整合所有產出
- 品質檢查
- 產出完整交付物

## 品牌設定

### 內容三大支柱與視覺
- **醫學**（藍色系 #3498DB）：專業知識、迷思破解
- **故事**（橘色系 A）：個人經歷、臨床故事
- **個人成長**（橘色系 B）：學習反思、習慣養成

### IG 格式
- **形式**：三張圖輪播（1080x1080px）
- **工具**：Canva（已有模板）
- **IG 帳號**：@momobear_doctor

### 共享知識庫
- `shared/visual-guidelines.md` - 視覺規範
- `shared/content-pillars.md` - 內容支柱定義
- `content-hub/ideas-pool.md` - 內容點子庫
- `resources/canva-templates.md` - Canva 模板
