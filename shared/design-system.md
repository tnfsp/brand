## 📱 設計結構

### 第 1 張：封面

#### 醫學主題封面
```
結構：
├─ 背景：AI 生成圖片（oil painting style），滿版
├─ 遮罩：藍色半透明（#005F73, 70-75%）
├─ 右上角方框：「每日醫學」（藍底白字）
├─ 中央：標題（大字，白色）
└─ 右下角：浮水印
```

#### 故事主題封面
```
結構：
├─ 背景：橘色純色（#CA6702 或 #EE9B00）
├─ 中間：AI 生成圖片（oil painting style）
├─ 下方：標題
└─ 右下角：浮水印
```

#### 個人成長主題封面
```
結構：
├─ 背景：AI 生成圖片（oil painting style），滿版
├─ 遮罩：橘黃色半透明（#CA6702, 70-75%）
├─ 標題框：
│   ├─ 黑色粗邊框（8-10px）
│   ├─ 背景：米黃色 #E9D8A6 或橘黃色 #EE9B00
│   └─ 文字：黑色，csong，60-70pt，粗體
├─ 副標題（可選）：放在黑框外
└─ 右下角：浮水印
```

**重點**：
- 封面圖片使用 AI 生成（oil painting style）
- 遮罩透明度：70-75%
- 標題要醒目、清楚

---

### 第 2-N 張：內容頁

#### 設計規範
```
結構：
├─ 背景：米黃色純色（#E9D8A6）
├─ 文字：
│   ├─ 字體：csong
│   ├─ 顏色：黑色
│   ├─ 標題：45-60pt，粗體
│   ├─ 內文：30-35pt，一般
│   └─ 對齊：左對齊或置中（視內容）
└─ 右下角：浮水印
```

**文字排版原則**：
- 行距：1.5-1.8
- 分段清楚（段落間留空行）
- 不要過度擁擠
- 標題與內文要有明顯層次

**內容類型**：
- 極簡型：單句話（50-60pt 大字）
- 有內文型：標題 + 列點或分段內文
- 混合型：部分極簡 + 部分有內文

---

### 最後一張：固定模板

#### 設計（不變動）
```
結構：
├─ 背景：米黃色 #E9D8A6
├─ 左右兩側：橘色條（#CA6702 或 #EE9B00）
├─ 上方：IG icon（粉紅漸層）
├─ 中間：固定文字
│   ├─ 「喜歡我的內容」
│   ├─ 「不一定需要按讚」
│   ├─ 「也不一定需要分享」
│   ├─ 「可以的話」
│   ├─ 「留個言跟我說些什麼」
│   └─ 「如果能跟你交個朋友」
│       「那就太好了 ☺」
└─ 右下角：浮水印
```

**重點**：
- 這張完全固定，不同貼文都用同樣設計
- 文字內容不變
- 直接套用模板即可

## 🖼️ AI 圖片生成指引

### 風格要求
- **風格**：Oil painting style（油畫風格）
- **色調**：溫暖色調（配合主題色系）
- **筆觸**：印象派風格，柔和、模糊
- **氛圍**：符合內容調性（專業/溫暖/反思）

### Prompt 範本

#### 醫學主題
```
Oil painting style, cool blue tones, [主題相關場景], professional and calm atmosphere, impressionist brushstrokes, muted colors, medical setting
```

#### 故事主題
```
Oil painting style, warm orange and golden tones, [故事相關場景], emotional and narrative atmosphere, impressionist brushstrokes, soft colors, human connection
```

#### 個人成長主題
```
Oil painting style, warm orange and golden tones, [成長相關場景], introspective and hopeful atmosphere, impressionist brushstrokes, muted colors, sense of growth and reflection
```

## 📋 製作流程

### Step 1：內容準備
1. 確認主題（醫學/故事/個人成長）
2. 準備文案（標題、內文、Caption）
3. 決定張數（建議 4-10 張）

### Step 2：生成封面圖片
1. 根據主題撰寫 AI 圖片 prompt
2. 使用 AI 工具生成圖片（Midjourney/DALL-E/Stable Diffusion）
3. 選擇最符合調性的圖片

### Step 3：在 Canva 製作
1. **封面**：
   - 插入 AI 圖片，滿版
   - 加上對應主題的半透明遮罩
   - 加上標題框（醫學/故事/成長的設計不同）
   - 右下角加浮水印
2. **內容頁**：
   - 米黃色背景 #E9D8A6
   - 使用 csong 字體
   - 黑色文字，注意字體大小與層次
   - 右下角加浮水印
3. **最後一張**：
   - 直接使用固定模板
   - 不需要更動

### Step 4：匯出
- 格式：PNG（高畫質）
- 尺寸：1080x1080px
- 檔名：`YYYY-MM-主題-標題-01.png`

## 💡 常見問題

### Q: 封面一定要用 AI 圖片嗎？
A: 是的，這是你的品牌視覺特色（oil painting style）。

### Q: 可以不用最後一張固定模板嗎？
A: 不建議。固定模板是品牌一致性的重要元素。

### Q: 內容頁可以用其他背景色嗎？
A: 建議統一使用米黃色 #E9D8A6，這是品牌識別色。

### Q: 字體可以換嗎？
A: 不建議。csong 是品牌視覺的一部分。

### Q: 張數有限制嗎？
A: IG 最多 10 張。建議 4-8 張為佳（太少不夠深度，太多受眾疲勞）。
