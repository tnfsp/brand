# Personal Inspiration - 個人靈感庫

> 這個資料夾存放 Wilson 從筆記軟體匯出的靈感庫檔案

---

## 📁 用途

這裡存放你從筆記軟體（Obsidian / Notion / Logseq 等）匯出的靈感 md 檔案。

Inspiration Agent 會讀取這些檔案，以便：
- ✅ 了解你過去記錄過哪些靈感
- ✅ 避免推薦重複的主題
- ✅ 識別你特別關注的主題類型
- ✅ 發現可以延伸或重新發展的舊點子

---

## 📝 如何使用

### 步驟 1：匯出你的靈感庫
從你的筆記軟體匯出為 Markdown 格式。

### 步驟 2：放到這個資料夾
將檔案放到這裡，例如：
```
content-hub/personal-inspiration/靈感庫.md
content-hub/personal-inspiration/Ideas-Vault.md
content-hub/personal-inspiration/想法收集.md
```

檔名隨意，可以是中文或英文。

### 步驟 3：讓 Agent 分析
執行 `/inspire` 時，Agent 會自動讀取這個資料夾中的檔案。

---

## 🔒 隱私保護

這個資料夾已加入 `.gitignore`，檔案**不會被提交到 Git**。

你的靈感內容保持私密，只存在本地端。

---

## 📋 建議的檔案結構

你可以：
- **單一檔案**：把所有靈感放在一個 md 檔案中
- **多個檔案**：依主題分類（醫學.md、故事.md、成長.md）
- **不限格式**：任何 markdown 格式都可以，Agent 會智能識別
