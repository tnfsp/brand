# 內容點子庫

> 這是所有內容主題的點子池。所有 Agents 在規劃內容時應先從這裡選擇，並標記使用狀態。

---

## 使用說明

### 點子格式
每個點子包含：
- **主題**：簡短描述
- **支柱**：醫學 / 故事 / 個人成長
- **適合渠道**：IG / Blog / 兩者皆可
- **優先順序**：高 / 中 / 低
- **狀態**：待使用 / 進行中 / 已發布
- **備註**：其他相關資訊

### 如何新增點子
- Leader 可隨時新增點子到對應的支柱分類
- Content-Framing Agent 也可以根據策略建議新點子
- 不用馬上完善所有細節，先記下來即可

### 如何使用點子
1. Content-Framing Agent 根據月度策略選擇適合的點子
2. 將狀態改為「進行中」
3. 產出完成後改為「已發布」，並加上發布日期與連結

## 點子發想提示

- 習慣養成的心得
- 跨領域學習的收穫

## 點子庫管理

### 定期檢視
- **每月初**：Content-Framing Agent 檢視點子庫，為本月選擇合適主題
- **每季**：Social Manager 與 Leader 檢視點子庫狀況，補充新點子

### 補充機制
- Leader 隨時可以新增點子（用手機記下來也可以，之後再整理）
- Social Manager 根據策略建議需要的點子方向
- Project Manager 在執行過程中發現新點子也可補充

### 優先順序調整
- 時事性強的主題提高優先順序
- 已經發布類似主題的降低優先順序
- 根據受眾反饋調整優先順序
